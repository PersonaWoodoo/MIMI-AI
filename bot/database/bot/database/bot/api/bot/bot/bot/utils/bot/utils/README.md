# 🤖 MiMi AI — Telegram Bot for AI Image & Video Generation

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)
[![Python 3.11](https://img.shields.io/badge/Python-3.11-blue.svg)](https://www.python.org/)

**MiMi AI** — это мощный Telegram бот для генерации изображений, видео, lip sync и кинематографического контента с использованием 200+ AI моделей (Flux, Midjourney, Kling, Sora и другие).

## ✨ Features

- 🎨 **Image Generation** — 50+ text-to-image models
- 🎬 **Video Generation** — 40+ text-to-video, 60+ image-to-video
- 🎙️ **Lip Sync** — animate portraits with audio (9 models)
- 🎥 **Cinema Studio** — camera controls, lenses, focal length
- 🔧 **Workflow Studio** — multi-step pipelines
- 📜 **Generation History** — store all your creations
- 💎 **Credit System** — track usage

## 🚀 Quick Start

### Prerequisites
- Python 3.11+
- Telegram Bot Token (from @BotFather)
- Muapi.ai API Key

### Installation

```bash
git clone https://github.com/yourusername/MiMi-AI-Bot.git
cd MiMi-AI-Bot
cp .env.example .env
# Edit .env with your tokens
docker-compose up -d
